About Us

We build an automation for accurate blood donor matching prediction that can be used in a hospitals.

To enhance blood donor–recipient matching accuracy using ML models.

This project proposes the development of a browser-based software application that leverages Machine Learning (ML) models such as K-Nearest Neighbour (KNN), Random Forest (RF) or Naïve Bayes, and Decision Tree to predict donor–recipient compatibility with higher accuracy.

  • To integrate Core medical factors in prediction (ABO, RhD, transfusion history, crossmatch results)..
  • To compare ML algorithms (KNN, RF, Naïve Bayes, Decision Tree) for best performance.
  • To provide a browser-based, role-specific hospital application.

We build an automation for accurate blood donor matching prediction that can be used in a hospitals. We propose a browser-based application that uses ML models (KNN, Random Forest, Naïve Bayes, Decision Tree) to predict donor–recipient compatibility based on core medical factors (ABO, RhD, crossmatch results, transfusion history, disease type) and extended features when available.
