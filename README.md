# Blood Donor–Recipient Compatibility Prediction System

## Overview

This project is a Machine Learning-based web application developed to predict blood donor–recipient compatibility. The system analyzes donor details, recipient details, and medical parameters to determine whether a blood transfusion is compatible or incompatible.

## Features

- Blood donor–recipient compatibility prediction
- Random Forest and Naïve Bayes algorithms
- Web-based user interface
- Fast and accurate prediction results
- Medical parameter analysis
- Decision support for healthcare professionals

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- MySQL
- Pandas
- NumPy
- Scikit-learn

## Dataset

The dataset contains 500 donor–recipient records with parameters such as:

- Blood Group
- Rh Factor
- Age
- Gender
- Hemoglobin Level
- Disease Type
- Previous Transfusions
- Crossmatch Result
- HIV Status
- HBsAg Status
- HCV Status
- Syphilis Status

## Machine Learning Models

| Algorithm | Accuracy |
|------------|------------|
| Random Forest | 96–97% |
| Naïve Bayes | 92–93% |

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Ashrith-555/blood-donor-recipient-compatibility-prediction.git
cd blood-donor-recipient-compatibility-prediction
```

2. Install required packages

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open in browser

```text
http://127.0.0.1:5000
```

## How to Use

1. Enter donor details.
2. Enter recipient details.
3. Provide medical parameters.
4. Click the Predict button.
5. View the compatibility result.
